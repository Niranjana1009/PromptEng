# AI-Generated Responsive Website

## Overview
This repository contains a responsive, single-page website generated using **Prompt Engineering** for a **Front-End Development Internship assignment**. The project demonstrates the creation of modern, clean, and professional UI components using **HTML** and **Tailwind CSS**.

The website includes the following sections:
- **Navigation Bar** – Responsive with a mobile hamburger menu.
- **Hero Section** – Headline, subtext, CTA button, and illustration.
- **Services / Features Section** – 3 service cards with icons and descriptions.
- **Contact Form** – Name, email, and message fields with styled inputs.
- **Footer** – Quick links, resources, and social media icons.

---

## Technologies Used
- **HTML5** – For structure and semantic layout.
- **Tailwind CSS (CDN)** – For modern, responsive, utility-first styling.
- **JavaScript** – Minimal use for mobile menu toggle.
- **Unsplash & Heroicons** – Free images and SVG icons.
